# نظام التنبؤ بمخاطر العملاء البنكية باستخدام الذكاء الاصطناعي
# AI-Powered Bank Customer Risk Prediction System

## 📌 مقدمة المشروع (Project Introduction)
هذا المشروع هو نموذج متقدم لمحاكاة نظام بنكي ذكي يعتمد على **التعلم الآلي (Machine Learning)** لتقييم المخاطر الأمنية والسيبرانية لكل عميل فردي. يهدف النظام إلى الكشف المبكر عن محاولات الاحتيال، وتقييم "النظافة الرقمية" لأجهزة العملاء، وتقديم توصيات أمنية دقيقة للبنك بناءً على سلوك العميل.

## 🌟 تميز العرض الأكاديمي (Academic Excellence)
تم تصميم هذا المشروع ليكون جاهزاً للعرض الجامعي المتميز، حيث يشمل:
1.  **لوحة تقييم الأداء (Model Evaluation Dashboard)**: عرض مصفوفة الارتباك (Confusion Matrix) ومقاييس الدقة (Precision, Recall, F1-Score).
2.  **تحليل العوامل (Feature Importance)**: توضيح العوامل الأكثر تأثيراً على قرار الذكاء الاصطناعي (مثل غياب 2FA أو فشل تسجيل الدخول).
3.  **بيانات ضخمة وهمية (Advanced Synthetic Data)**: توليد بيانات واقعية لـ 2000 عميل تحاكي سيناريوهات الهجمات السيبرانية الحقيقية.
4.  **تصدير تقارير (Report Exporting)**: القدرة على استخراج تقرير أمني لكل عميل بصيغة نصية احترافية.

## 🛠️ التقنيات والمكتبات المستخدمة
*   **النموذج الذكي**: Random Forest Classifier (خوارزمية الغابات العشوائية) - المتميزة بالدقة العالية في البيانات الجدولية.
*   **الواجهة الرسومية**: `ttkbootstrap` لتقديم واجهة عصرية تتناسب مع هوية البنوك الرقمية.
*   **التحليل البياني**: `Matplotlib` و `Seaborn` لرسم الخرائط الحرارية والرسوم التوضيحية.
*   **معالجة البيانات**: `Pandas` و `NumPy`.

## 🛡️ العوامل الأمنية والسيبرانية في النموذج
يقيم النظام 11 عاملاً مختلفاً، منها عوامل سيبرانية بحتة:
*   **المصادقة الثنائية (Two-Factor Authentication)**: هل العميل محمي ضد سرقة الحساب؟
*   **عمر كلمة المرور (Password Age)**: هل يتبع العميل سياسات تغيير كلمة المرور الدورية؟
*   **سجل البرمجيات الخبيثة (Malware History)**: هل تم رصد إصابات سابقة بجهاز العميل؟
*   **تغيير الموقع المشبوه (Impossible Travel)**: الكشف عن تسجيل الدخول من دولتين مختلفتين في وقت قياسي.

## 📂 هيكل الملفات
*   `main.py`: المشغل الرئيسي للنظام.
*   `gui_app.py`: كود الواجهة الرسومية المتقدمة (نظام التبويبات).
*   `model_logic.py`: العقل المدبر للنظام (بناء وتقييم النموذج).
*   `data_generator.py`: محاكي بيانات العملاء والأنماط السلوكية.

## 🚀 كيفية التشغيل
1. تأكد من تثبيت المكتبات اللازمة:
   ```bash
   pip install pandas numpy scikit-learn matplotlib seaborn ttkbootstrap
   ```
2. قم بتشغيل الملف الرئيسي:
   ```bash
   python main.py
   ```
