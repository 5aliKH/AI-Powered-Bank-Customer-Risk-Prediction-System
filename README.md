# AI-Powered Bank Customer Risk Prediction System

## Overview

AI-Powered Bank Customer Risk Prediction System is an intelligent banking security platform that leverages Machine Learning to assess customer cyber-risk levels, detect suspicious behavior patterns, and support proactive fraud prevention strategies.

The system analyzes multiple behavioral, financial, and cybersecurity-related factors to generate risk predictions and provide actionable security recommendations for banking institutions.

This project demonstrates how Artificial Intelligence can be integrated into modern banking environments to strengthen cybersecurity, improve fraud detection, and enhance customer protection.

---

## Key Features

### Customer Risk Assessment

* Evaluate customer security risk levels
* Predict potential fraudulent activities
* Generate automated security recommendations

### Machine Learning Engine

* Random Forest Classifier model
* High prediction accuracy
* Feature importance analysis
* Automated risk categorization

### Cybersecurity Monitoring

* Two-Factor Authentication (2FA) assessment
* Password age monitoring
* Malware history analysis
* Suspicious login detection
* Impossible travel detection

### Interactive Dashboard

* Modern graphical interface using ttkbootstrap
* Easy navigation and visualization

### Reporting System

* Individual customer security reports
* Professional exportable reports
* Risk summaries and recommendations
